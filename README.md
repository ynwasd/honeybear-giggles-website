# Honeybear Giggles Website

Official website for comedian Honeybear Giggles — comedy that stings just right.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>
cd honeybear-giggles-website

# Step 2: Install dependencies
npm install

# Step 3: Start the development server
npm run dev
```

Requires Node.js and npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating) if needed.

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run preview` — Preview production build locally
- `npm run lint` — Run ESLint

## Deployment

This project is configured for GitHub Pages. Push to `main` or `master` to trigger automatic deployment via GitHub Actions.

## Content

Update tour dates, merch, media, and copy in `src/data/content.ts`.
